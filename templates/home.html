SciContext: JSON-LD context file creation for all

Semantic data is becoming an important part of the data publising landscape. Any encoding of RDF (Resource Description Framework) requires that you annotate your data, so it can be converted into RDF triples - subject, predicate, object statements of knowledge. For the JSON for linked data encoding (JSON-LD) of RDF, semantic annotation of data is provided through the inclusion of context files. If you are developing JSON-LD files this can be an issue as not only do you need to create them, you need to make them available online so the conversion to RDF can be done. This Django project can help you create, manage, and deploy any/all context files you need for your application(s).
